# Study Time Tracker
This program is a Python program that helps learners track how many hours they study in a week based on number of hours they study each day. This program will ask the users for daily study hours, then convert the input and calculate hours they study the whole week. It also includes error handling to ensure that users fill in accurate type of hours.

# How to run
python app.py

# Example Output
Enter hours studied today: 3
You are on track to study 21 hours this week.
